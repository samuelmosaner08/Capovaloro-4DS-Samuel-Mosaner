Aivia Platform

Aivia è una piattaforma web full-stack per il monitoraggio urbano in tempo reale. Centralizza dati provenienti da sensori eterogenei — telecamere di sorveglianza, sensori meteo, semafori intelligenti, contatori di persone, monitor di parcheggi e bidoni — in un'unica dashboard interattiva e configurabile. Aivia is a full-stack web platform for real-time urban monitoring. It centralizes data from heterogeneous sensors — surveillance cameras, weather stations, smart traffic lights, people counters, parking monitors and trash bins — in a single interactive, configurable dashboard.

ArchitetturaArchitecture

Architettura del SistemaSystem Architecture

Il sistema è composto da tre layer distinti: un frontend Next.js, un backend Node.js con tre server concorrenti, e un database PostgreSQL. L'architettura separa le responsabilità: GraphQL per le query, REST per l'ingestion dei sensori, Socket.IO per il real-time. The system has three distinct layers: a Next.js frontend, a Node.js backend running three concurrent servers, and a PostgreSQL database. The architecture separates concerns: GraphQL for queries, REST for sensor ingestion, Socket.IO for real-time.

Query di campionamento metricheMetric sampling query

Il backend aggrega le letture dei sensori in finestre temporali (5-60 minuti) prima di restituirle al frontend. The backend aggregates sensor readings into time windows (5-60 minutes) before returning them to the frontend. 

-- Campionamento metriche con aggregazione oraria (24 ore)
SELECT
    widget_id,
    AVG(value::numeric)              AS media_valore,
    DATE_TRUNC('hour', created_on)   AS ora
FROM metrics
WHERE
    widget_id = $1
    AND created_on >= NOW() - INTERVAL '24 hours'
    -- Il QueryBuilder aggiunge automaticamente:
    -- AND deleted_on IS NULL
GROUP BY widget_id, ora
ORDER BY ora ASC;

Su mobile il fullscreen è implementato con un CSS overlay position:fixed come fallback e screen.orientation.lock('landscape') per bloccare l'orientamento.On mobile, fullscreen is implemented with a CSS overlay position:fixed as fallback and screen.orientation.lock('landscape') to lock orientation.

// Fullscreen mobile con orientamento paesaggio
async function enterFullscreen(containerEl) {
  if (document.fullscreenEnabled) {
    await containerEl.requestFullscreen();
  } else {
    // Fallback CSS overlay (stile YouTube)
    containerEl.classList.add('fullscreen-overlay');
  }
  // Blocca orientamento landscape (Android)
  if (screen.orientation?.lock) {
    try {
      await screen.orientation.lock('landscape');
    } catch (_) {
      // iOS non supporta orientation.lock
    }
  }
}
